
The files in this directory provide a working example of the INDS XML based on
Bill Finger's schema.

P3_startup_raw.txt: an overview of the commands used to launch the P3 INDS
                    system

P3_startup.xml: XML instance file describing the P3 INDS system

inds_startup.xsd: a modified version of Bill's schema; contains changes needed
                  in order to describe the P3 INDS system
